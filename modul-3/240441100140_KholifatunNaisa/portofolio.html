Tentang Saya

Halo! Saya Kholifatun Naisa, mahasiswa aktif di bidang Sistem Informasi yang memiliki ketertarikan besar dalam pengembangan aplikasi, analisis data, dan pemrograman. Saya senang belajar hal-hal baru, mengeksplorasi ide kreatif, dan mengembangkan diri melalui akademik maupun personal. Meskipun saat ini saya masih dalam tahap studi, saya berkomitmen untuk terus mengasah keterampilan saya dan membangun portofolio yang solid untuk masa depan. Semenjak di Madrasah Aliyah Negeri Sumenep, saya aktif mengikuti ekstrakurikuler seperti Robotika dan English Club, sementara beberapa lainnya tidak saya lanjutkan. Saya juga bergabung dalam organisasi OSIM selama dua periode, yang memberi saya banyak pengalaman berorganisasi. Selain itu, saya memiliki pengalaman dalam berbagai kompetisi, pelatihan, dan penghargaan, seperti peserta lomba Robot Cerdas, Pemrograman Excel, Finalis Lomba Presentasi Tiga Menit, dan Kreativitas Teknologi Remaja, serta sertifikat di Excel Data Analysis dan program pelatihan Samsung Innovation Campus. Saya juga menerima penghargaan sebagai Siswa Prestasi Akademik di jurusan Bahasa.
